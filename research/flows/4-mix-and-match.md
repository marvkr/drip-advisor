# Flow 4: Mix & Match

**DripAdvisor feature:** Combine tops and bottoms from different brands to build outfits.

## Best References

### Whering — `whering-ios/`
This is literally their core feature.
- `creating-an-outfit` (10 shots) — combining items into an outfit
- `outfit-detail` (5 shots) — viewing a combined outfit
- `shuffling-existing-outfits` (7 shots) — randomizing combinations
- `outfits` (2 shots) — outfits grid view
- `creating-a-lookbook` (7 shots) — collecting outfits into lookbooks
- `lookbooks` (2 shots) — lookbook browsing
- `creating-a-moodboard` (6 shots) — visual mood board creation
- `moodboards` (5 shots) — mood board grid

### Pinterest — `pinterest-ios/`
Collage creation is conceptually similar to mixing items.
- `creating-a-collage` (18 shots) — combining multiple images into one
- `saving-to-collage` (3 shots) — adding items to existing collage
- `start-new-collage` (3 shots) — collage creation entry point

### Nike — `nike-ios/`
- `complete-the-look` (2 shots) — outfit completion suggestions
- `how-others-are-wearing-it` (5 shots) — outfit context / styling inspiration

### FARFETCH — `farfetch-ios/`
- `complete-the-look` (3 shots) — outfit pairing suggestions
- `style-essentials` (6 shots) — curated outfit combinations

## Key Patterns to Study
- Whering's outfit creation (selecting top + bottom + accessories from wardrobe)
- Whering's shuffle feature (quick way to try random combinations)
- Pinterest's collage builder (layout tools for combining visuals)
- "Complete the look" patterns from Nike/FARFETCH (algorithmic pairing suggestions)
